# Output Directory Structure

This directory contains all outputs generated by the MATLAB scripts in this repository.

## Directory Organization

```
output/
├── vertices/       # Vertex files (.vertex)
├── images/         # All plot images (.png)
├── pdfs/          # PDF files (if generated)
└── data/          # Data files (.dat, .txt)
```

## Output Files by Script

### 1. `eel2d_straightswimmer.m`
- **Vertex file**: `vertices/eel2d.vertex`
- **Plot**: `images/eel2d_straightswimmer.png`

### 2. `naca_comparison/naca_method_comparison.m`
- **Plots**: `images/comparison_NACA*.png` (multiple files, one per thickness)
  - `comparison_NACA0004.png`
  - `comparison_NACA0008.png`
  - `comparison_NACA0012.png`
  - `comparison_NACA0016.png`
  - `comparison_NACA0020.png`
  - `comparison_NACA0024.png`

### 3. `naca_comparison/validation_test.m`
- **Plot**: `images/validation_test_NACA0012.png`

### 4. `naca_comparison/naca_original_method.m`
- **Plots**: `images/NACA*_original.png` (multiple files, one per thickness)
- **Data files**: `data/NACA*_coords.dat` (airfoil coordinates)
- **Data files**: `data/NACA*_area.txt` (area analysis)

## Usage

All MATLAB scripts will automatically create these directories if they don't exist. Simply run any script and outputs will be organized in the appropriate subdirectories.

## Notes

- All output files are ignored by git (see `.gitignore`)
- The organized structure makes it easy to find specific types of outputs
- Vertex files are used for IBAMR simulations
- Image files can be used for documentation and analysis
- Data files contain numerical results and coordinates
